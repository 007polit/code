图标设计特点：
--------------------------------------------------------------------------------
- 风格：现代扁平化设计
- 主色：蓝紫渐变（#4A5FFF → #8B5CF6）
- 字母：斜体 E（代表 Editer）
- 效果：圆形背景 + 阴影 + 高光 + 白色描边
- 尺寸：支持 16x16 到 256x256 多种尺寸

使用步骤：
--------------------------------------------------------------------------------
1. 运行图标生成脚本：
   cd assets
   generate_icon.bat

2. 重新编译项目：
   cd ..
   build_msvc.bat

3. 查看结果：
   在资源管理器中查看 editer.exe 的图标

技术实现：
--------------------------------------------------------------------------------
- SVG 矢量图设计（可无损缩放）
- Python + Pillow + CairoSVG 自动生成多尺寸 ICO
- Windows RC 资源脚本嵌入图标
- MSVC 编译时自动链接资源文件
