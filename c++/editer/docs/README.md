# Editer - 终端文本编辑器

一个功能强大的终端文本编辑器，支持文件管理、语法高亮、AI 辅助和集成终端。

## ✨ 主要功能

- **Vim 风格编辑** - 支持 Normal、Insert、Visual、Command 模式
- **自动保存** - 可配置间隔的自动保存功能，防止数据丢失
- **文件树管理** - 完整的文件和文件夹操作，支持剪切/复制/粘贴
- **语法高亮** - 支持 C++、Python、Markdown、JSON、CFG
- **AI 代码助手** - 基于远端 OpenAI 风格 API 的智能对话
- **集成终端** - 屏幕底部 5 行固定高度终端
- **配置系统** - 基于 DSL 的配置文件，支持自定义设置
- **智能编辑** - 自动缩进、括号匹配、撤销/重做

## 🚀 快速开始

### 安装

详细安装说明请参见 [INSTALL.md](INSTALL.md)。

```bash
# 克隆仓库
git clone <repository-url>
cd editer

# 使用 MSVC 构建 (Windows)
build_msvc.bat

# 运行编辑器
editer.exe
editer.exe examples/test.cpp
```

### 基本使用

```bash
# 打开文件
editer.exe myfile.cpp

# 创建新文件
editer.exe newfile.txt
```

## 🤖 AI 助手

Editer 通过远端 OpenAI 风格的 HTTP API 提供 AI 代码辅助功能（例如 `https://llmapi.paratera.com/v1`）。

### 设置

1. 在 `config.cfg` 中配置：
   - `api_url`（例如 `https://llmapi.paratera.com`）
   - `api_key`（你的 API 密钥）
   - `api_model`（例如 `DeepSeek-R1`）

### 使用

| 按键 | 功能 | 说明 |
|-----|------|------|
| `Ctrl+A` | 切换 | 打开/关闭 AI 聊天 |
| `↑/↓` | 滚动 | 滚动聊天历史 |
| `Enter` | 发送 | 发送消息给 AI |
| `ESC` | 关闭 | 关闭 AI 聊天 |

### 推荐模型

| 模型 | 适用场景 |
|------|----------|
| `DeepSeek-R1` | 通用对话、代码解释与辅助 |
| `Qwen3-Coder-480B-A35B-Instruct` | 偏重代码相关对话 |

## ⚙️ 配置

编辑项目根目录的 `config.cfg`:

```cfg
# Tab 大小 (每个 Tab 的空格数)
set tabsize = 2

# 自动保存 (秒数, 0 禁用)
set auto_save = 5
```

**支持的设置:**

| 设置项 | 类型 | 默认值 | 范围 | 说明 |
|--------|------|--------|------|------|
| `tabsize` | int | 4 | 1-16 | Tab 键插入的空格数 |
| `auto_save` | int | 5 | 0-无限 | 自动保存间隔（秒），0 表示禁用 |

### 自动保存说明

- **工作原理**: 编辑文件后，编辑器会在指定的时间间隔内自动保存文件
- **仅在有改动时保存**: 如果文件未被修改，不会触发保存
- **手动保存优先**: 按 `Ctrl+S` 或命令 `:w` 手动保存会重置自动保存计时器
- **日志记录**: 自动保存事件会记录在日志中，但不会在界面显示提示
- **文件必须打开**: 仅对已打开的文件生效，新建未命名文件需先用 `:w filename` 保存

**示例配置:**

```cfg
# 启用 5 秒自动保存
set auto_save = 5

# 启用 10 秒自动保存
set auto_save = 10

# 禁用自动保存
set auto_save = 0
```

编辑器会自动将 `config.cfg` 转换为 `config.json` 以加快加载速度。

## 🐛 故障排除

### AI 聊天不可用

1. 检查 `config.cfg` 中的 `api_url`、`api_key`、`api_model` 是否填写正确
2. 在终端中使用 `curl` 测试 API 是否可用
3. 查看 `editer.log` 中与 AI 相关的错误日志（HTTP 状态码、错误信息）

### 终端不显示

- 终端可见时始终占据底部 5 行
- 按 `Ctrl+T` 切换
- 检查终端是否被其他窗口遮挡

### 文件树问题

- 确保有适当的文件权限
- 检查目录是否可读
- 尝试按两次 `Ctrl+F` 刷新
